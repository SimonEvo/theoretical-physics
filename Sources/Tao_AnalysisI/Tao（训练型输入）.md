目标：恢复 rigor + epsilon thinking

### 使用方式：

每一章拆成三层：

### (1) Reading notes（Source layer）

```
Sources/Tao_Analysis/chapter1.md
```

写：

- 定义
- theorem
- proof outline
- 卡住点

---

### (2) Concept extraction（Concept layer）

例如：

```
Concepts/    Limits_and_Continuity.md    Metric_Spaces.md    Completeness.md
```

---

### (3) Failure log（非常关键）

专门记录：

```
Problems/Tao_failures.md
```

写：

- 哪一步没看懂
- 为什么没看懂
- 重新理解版本

这一步比“看懂”更重要。
